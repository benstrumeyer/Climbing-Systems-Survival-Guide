# Climbing Systems

A personal climbing coach focused on safety systems and training for big wall climbing — ultimate goal: El Capitan.

## Scope

Covers lead climbing, trad climbing, bouldering, multi-pitch systems, anchors, knots, and training progressions. Sourced from curated books and web resources (user-directed).

## Goals

- Teach safe outdoor climbing systems (knots, anchors, belaying, rope management)
- Generate lessons and structured learning paths
- Build a training plan to climb El Capitan
- Current climber level: V9 boulderer
